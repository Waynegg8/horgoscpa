## 🚀 未來執行重構的標準指令

### 📌 首次讓 AI 工作（必須先執行）

```
你好！請先閱讀以下文檔了解專案規範：



核心規範：
✅ 任何修改必須先更新設計文檔
✅ 遇到問題必須深度診斷（五個為什麼），找根本原因
✅ 樣式問題優先修改組件庫而非單一頁面
✅ 代碼重複必須提取到共用模組
✅ 資料庫必須遵循命名規範（{table}_id, is_{name}, {action}_at）
✅ API 必須 100% RESTful
✅ 前端使用 Core-Components-Modules-Pages 架構
✅ 後端使用 Repository-Service-Handler-Routes 架構
❌ 絕對禁止表面修復
❌ 絕對禁止生成任何總結報告

準備好後開始工作。
```
