# 09 - 加權工時計算

**所屬模塊：** 工時表與報表  
**最後更新：** 2025年10月27日

---

## 功能概述

根據加班費率自動計算加權工時。

---

## 計算邏輯

```typescript
function calculateWeightedHours(workType: string, hours: number) {
  const rate = getRate(workType);  // 從 OvertimeRates 表查詢
  return hours * rate;
}
```

**範例：**
- 平日加班(1.34) 2小時 → 2.68 加權工時

---

## 資料表

`OvertimeRates`
