# 12 - 生活事件登記

**所屬模塊：** 假期管理  
**頁面路徑：** `/leave/events`  
**權限：** 所有使用者  
**最後更新：** 2025年10月27日

---

## 功能概述

員工自主登記婚假、喪假等事件，系統據此計算假期額度。

---

## UI 設計

```
生活事件登記
───────────────────────
事件類型：[婚假 ▼]
事件日期：[2025-12-25]

[送出]

已登記事件：
• 2023-05-10  結婚  (已產生 8 天婚假額度)
```

---

## 資料表

`LeaveEvents`

---

## API

```
POST /api/v1/leave/events
GET  /api/v1/leave/events
```
