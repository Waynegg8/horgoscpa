# ServiceTemplates (服務模板)

**最後更新：** 2025年10月27日

---

## 表格定義

```sql
CREATE TABLE ServiceTemplates (
  template_id INTEGER PRIMARY KEY AUTOINCREMENT,
  name TEXT NOT NULL,
  description TEXT
);
```

---

## 欄位說明

| 欄位 | 類型 | 說明 |
|------|------|------|
| `template_id` | INTEGER PK | 服務模板 ID |
| `name` | TEXT | 服務名稱（例如：'記帳服務', '工商登記服務'） |
| `description` | TEXT | 服務說明 |
