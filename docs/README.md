## 📝 文檔規範

所有新增的說明文檔請統一放置在此 `docs/` 資料夾中：

### 一般文檔
```
docs/
  └── YOUR_DOCUMENT.md
```

### API 或子系統相關文檔
```
docs/
  └── [子系統名稱]/
      └── YOUR_DOCUMENT.md
```

### 文件命名規則
- 使用大寫字母和底線：`EXAMPLE_GUIDE.md`
- 使用英文命名（可在內容中使用中文）
- 使用描述性的名稱，如：
  - `SETUP_GUIDE.md` - 設定指南
  - `USER_MANUAL.md` - 使用手冊
  - `API_REFERENCE.md` - API 參考
  - `TROUBLESHOOTING.md` - 疑難排解
