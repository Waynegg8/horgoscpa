# 自動化｜每季自動執行

## 季度任務自動生成（每季第一天 00:00）

### 流程
```
1) 掃描服務：週期 = 每季；狀態 = 啟用中
2) 生成季度任務（如：Q1 稅務申報）
3) 通知負責員工
```

### 範例（2026/1/1）
```
- ABC科技 - Q1稅務申報（王小明）
- XYZ公司 - Q1財務報表（李美玲）
... 共 8 個
```

---

## 季報生成（每季第一個月 5 日 08:00）

### 內容
```
- 季度營收統計
- 季度成本分析
- 季度盈利分析
- 客戶服務統計
- 員工績效統計
```

### 輸出
- Email 寄送管理員，作為經營決策參考
