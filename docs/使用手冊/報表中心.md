# 報表中心

**功能說明：** 查看工時、假期等統計報表  
**誰可以用：** 所有人（員工看自己，管理員看全部）

---

## 🎯 這個功能做什麼

查看各種統計報表，幫助了解：
- 本月工作了多少時間
- 假期使用情況
- 各客戶工時分布
- 工時趨勢變化

---

## 📊 可用的報表

### 工時統計報表
- 總工時
- 加權工時
- 各客戶工時分布
- 各服務項目工時分布
- 工時趨勢圖（折線圖）

### 假期統計報表
- 各假別使用情況
- 假期餘額狀態
- 假期使用趨勢

### 客戶工時報表（管理員）
- 各客戶的總工時
- 各客戶的費用統計
- 員工工時分布

---

## 📱 如何使用

1. 點選左側選單「報表中心」
2. 選擇日期範圍（本週/本月/本季/自訂）
3. 選擇報表類型
4. 系統自動產生報表
5. 可以匯出 Excel 或 PDF
