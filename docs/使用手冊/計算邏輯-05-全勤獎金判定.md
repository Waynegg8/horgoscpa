# 計算邏輯｜全勤獎金判定（3.5）

## 規則

### 不影響全勤
- 補休、特休、公傷病假、婚假、喪假、公假、產假、產檢假、陪產檢及陪產假、家庭照顧假、生理假（半薪）

### 會扣除全勤
- 病假（普通傷病假）
- 事假

### 生理假特殊規則
- 生理假不影響全勤，但給半薪
- 全年前 3 日不併入病假；第 4 日起併入病假計算

---

## 實際範例

### 符合全勤
```
10 月出勤：補休 1 天、特休 1 天，其餘正常 → ✅ 符合
全勤獎金：給予
```

### 不符合全勤（病假）
```
10 月出勤：病假 0.5 天、補休 1 天 → ❌ 不符合（因病假）
```

### 不符合全勤（事假）
```
10 月出勤：事假 1 天、特休 1 天 → ❌ 不符合（因事假）
```
