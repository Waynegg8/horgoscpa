# 計算邏輯｜工時與補休計算（3.1）

## 基本概念
- 實際工時：員工真正工作的時間
- 加權工時：僅用於計算加班費（實際工時 × 費率倍數）
- 補休時數：與實際加班時數 1:1 對應

## 計算公式
```
加班費用加權工時 = 實際工時 × 費率倍數
補休時數 = 實際加班時數 × 1（1:1 對應）
每日工時上限 = 12 小時
```

⚠️ 補休是 1:1 對應，不是按加權工時計算。

---

## 實際範例

### 範例 1：正常工時
```
日期：2025-11-04（週一）
類型：正常工時（1.0 倍）
工時：8 小時

計算：
- 實際工時：8
- 加權工時：8 × 1.0 = 8
- 補休：0 小時
```

### 範例 2：平日加班（第 9-10 小時）
```
工時：2 小時；費率：1.34 倍
計算：
- 實際加班：2
- 加班費計算：2 × 時薪 × 1.34
- 補休：2 小時（1:1），到期：該月底
```

### 範例 3：休息日加班（分段費率）
```
工時：6 小時；分段：前2小時 1.34 倍，3-6小時 1.67 倍
計算：
- 實際加班：6
- 加班費：2×1.34 + 4×1.67（乘以時薪）
- 補休：6 小時（1:1），到期：該月底
```

### 範例 4：國定假日加班
```
工時：4 小時（國慶日）
計算：
- 實際加班：4
- 加班費：8 小時工資（8小時內統一）
- 補休：8 小時（統一產生，與加班費對等）
```
