## 🎯 使用流程

### 用户侧
```
1. 看一次 "如何使用AI開發.md"（3分钟）
2. 之后每次只需要说需求
3. 回答AI的业务问题
4. 等待完成，直接使用
```

### AI 侧
```
1. 开始任务前读取 3 个核心文档
2. 查阅决策树做技术决策
3. 查阅词典理解业务术语
4. 完全自主完成所有工作
5. 确认无误后报告完成
```
