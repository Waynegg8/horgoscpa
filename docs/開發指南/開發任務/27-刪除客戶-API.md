# 開發任務 27：刪除客戶 API（軟刪除）

**對應規格：** 後端 `docs/開發指南/後端/客戶管理-後端規格.md`

**對應手冊：** `docs/使用手冊/功能模組-02-客戶管理.md`

## 範圍

實作 `DELETE /api/v1/clients/:id` API，軟刪除客戶。

## 輸入

- `client_id`（路徑參數）

## 輸出

- 成功訊息

## 驗證與權限

- 需登入
- 員工可刪除所有客戶（小型事務所彈性設計）
- 客戶不存在或已刪除回傳 404

## 實作重點

- 設置 `is_deleted = 1`
- 記錄 `deleted_at` 和 `deleted_by`
- 不實際刪除資料（軟刪除）
- 後續查詢需過濾 `is_deleted = 0`

## 驗收標準

- [ ] API 正確執行軟刪除
- [ ] 記錄刪除稽核資訊
- [ ] 已刪除的客戶不會出現在列表中
- [ ] 已部署並可透過前端呼叫
