## 約定與共用內容（減少重複）

- UI 設計文檔通用模板：頁面結構、主要元件、交互流程、空狀態/錯誤狀態。
- 業務邏輯文檔通用模板：角色前置條件、流程圖、規則列表、邊界條件、錯誤碼。
- 測試案例通用模板：前置數據、步驟、期望結果、對應 API、對應資料表。
- 共用附錄：
  - 分頁/排序/篩選規範
  - 統一錯誤回應格式
  - 權限檢查規則（參考：架構設計/權限系統設計）
