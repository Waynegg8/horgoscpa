# 刪除任務模板

**端點：** `DELETE /api/v1/task-templates/:id`  
**權限：** 管理員

---

## 請求

### 路徑參數
| 參數 | 類型 | 必填 | 說明 |
|-----|------|------|------|
| `id` | integer | ✅ | 模板ID |

---

## 成功回應

**HTTP 狀態碼：** 200

```json
{
  "success": true,
  "data": {
    "message": "任務模板刪除成功"
  }
}
```

---

## 錯誤回應

### 模板正在使用
**HTTP 狀態碼：** 409
```json
{
  "success": false,
  "error": {
    "code": "TEMPLATE_IN_USE",
    "message": "此模板正被客戶服務使用，無法刪除"
  }
}
```

### 有進行中任務
**HTTP 狀態碼：** 409
```json
{
  "success": false,
  "error": {
    "code": "TEMPLATE_HAS_ACTIVE_TASKS",
    "message": "此模板有進行中的任務，無法刪除"
  }
}
```
