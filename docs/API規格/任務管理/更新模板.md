# 更新任務模板

**端點：** `PUT /api/v1/task-templates/:id`  
**權限：** 管理員

---

## 請求

### 路徑參數
| 參數 | 類型 | 必填 | 說明 |
|-----|------|------|------|
| `id` | integer | ✅ | 模板ID |

### Body
```json
{
  "name": "記帳標準流程 v2.0",
  "description": "更新後的流程說明",
  "stages": [
    {
      "stage_name": "資料收集與核對",
      "estimated_days": 3,
      "depends_on": null
    },
    {
      "stage_name": "營業稅過帳",
      "estimated_days": 3,
      "depends_on": 0
    }
  ]
}
```

**說明：**
- 只需傳送要更新的欄位
- 若提供 `stages`，將完全替換現有階段

---

## 成功回應

**HTTP 狀態碼：** 200

```json
{
  "success": true,
  "data": {
    "message": "任務模板更新成功"
  }
}
```

---

## 錯誤回應

### 模板不存在
**HTTP 狀態碼：** 404
```json
{
  "success": false,
  "error": {
    "code": "TEMPLATE_NOT_FOUND",
    "message": "任務模板不存在"
  }
}
```
