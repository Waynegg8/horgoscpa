# 查詢任務模板詳情

**端點：** `GET /api/v1/task-templates/:id`  
**權限：** 管理員

---

## 請求

### 路徑參數
| 參數 | 類型 | 必填 | 說明 |
|-----|------|------|------|
| `id` | integer | ✅ | 模板ID |

---

```json
  "success": false,
  "error": {
    "code": "TEMPLATE_NOT_FOUND",
    "message": "任務模板不存在"
  }

```
