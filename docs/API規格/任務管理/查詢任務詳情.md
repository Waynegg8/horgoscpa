# 查詢任務詳情

**端點：** `GET /api/v1/tasks/:id`  
**權限：** 所有員工

---

## 請求

### 路徑參數
| 參數 | 類型 | 必填 | 說明 |
|-----|------|------|------|
| `id` | integer | ✅ | 任務ID |

---

{
  "success": false,
  "error": {
    "code": "TASK_NOT_FOUND",
    "message": "任務不存在"
  }
