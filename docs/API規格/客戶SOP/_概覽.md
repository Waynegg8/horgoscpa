# 客戶SOP API 概覽

**API 前綴：** `/api/v1`  
**最後更新：** 2025年10月27日

---

## 端點總覽

| 端點 | 方法 | 權限 | 說明 |
|------|------|------|------|
| `/clients/:id/sop` | GET | 所有員工 | 查詢客戶的 SOP 列表 |
| `/clients/:id/sop` | POST | 管理員 | 關聯 SOP 到客戶 |
| `/clients/:id/sop/:linkId` | DELETE | 管理員 | 解除 SOP 關聯 |
