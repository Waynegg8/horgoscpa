# 查詢客戶詳情

**端點：** `GET /api/v1/clients/:client_id`  
**權限：** 所有員工（非管理員只能查看自己負責的）

---

## 路徑參數

| 參數 | 類型 | 必填 | 說明 |
|-----|------|------|------|
| `client_id` | string | ✅ | 客戶統一編號 |

---

## 請求範例

```bash
GET /api/v1/clients/86753078
```

---

## 錯誤回應

### 客戶不存在
**HTTP 狀態碼：** 404
```json
{
  "success": false,
  "error": {
    "code": "CLIENT_NOT_FOUND",
    "message": "客戶不存在"
  }
}
```

### 無權限查看
**HTTP 狀態碼：** 403
```json
{
  "success": false,
  "error": {
    "code": "FORBIDDEN",
    "message": "您無權查看此客戶"
  }
}
```

---

## 業務邏輯

### 權限檢查
1. 管理員：可查看所有客戶
2. 員工：只能查看自己負責的客戶（`assignee_user_id` 匹配）

### 返回資料
- 包含所有客戶基本資料
- 包含關聯的標籤列表
- `registered_business_items` 為 JSON 陣列
