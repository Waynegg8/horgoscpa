# 加權工時 API 概覽

**API 前綴：** `/api/v1`  
**最後更新：** 2025年10月27日

## 端點總覽

| 端點 | 方法 | 權限 | 說明 |
|------|------|------|------|
| `/work-types` | GET | 所有員工 | 查詢工時類型列表 |
| `/work-types/calculate` | POST | 所有員工 | 計算加權工時 |
