# 刪除規則

**端點：** `DELETE /api/v1/settings/other-leave-rules/:id`  
**權限：** 管理員  
**最後更新：** 2025年10月27日

---

## 概述

刪除特定事件型假期規則。

---

## 請求

### 路徑參數

- `id` (必填, integer): 規則 ID

---

## 回應

### 成功回應（HTTP 200）

```json
{
  "success": true,
  "message": "假期規則已刪除",
  "data": {
    "deleted_rule_id": 10
  }
}
```

---

### 錯誤回應

**404 Not Found：**
```json
{
  "success": false,
  "error": {
    "code": "OTHER_LEAVE_RULE_NOT_FOUND",
    "message": "找不到指定的假期規則"
  }
}
```
