# 業務規則 API 概覽

**API 前綴：** `/api/v1`  
**最後更新：** 2025年10月27日

---

## 端點總覽

### 國定假日管理

| 端點 | 方法 | 權限 | 說明 |
|------|------|------|------|
| `/rules/holidays` | GET | 管理員 | 查詢國定假日列表 |
| `/rules/holidays` | POST | 管理員 | 新增國定假日 |
| `/rules/holidays/:id` | PUT | 管理員 | 更新國定假日 |
| `/rules/holidays/:id` | DELETE | 管理員 | 刪除國定假日 |
| `/rules/holidays/bulk-import` | POST | 管理員 | 批量導入假日 |
