# CSV導入 API 概覽

**API 前綴：** `/api/v1`  
**最後更新：** 2025年10月27日

---

## 端點總覽

| 端點 | 方法 | 權限 | 說明 |
|------|------|------|------|
| `/import/templates/:type` | GET | 管理員 | 下載 CSV 範本 |
| `/import/validate` | POST | 管理員 | 驗證 CSV 資料 |
| `/import/execute` | POST | 管理員 | 執行批量導入 |

---

## 支援的導入類型

- `clients` - 客戶資料
- `services` - 服務項目
- `employees` - 員工帳號
- `client_services` - 客戶服務設定
