# 更新個人資料

**端點：** `PUT /api/v1/profile`  
**權限：** 所有員工

---

## 請求

### Body
```json
{
  "name": "張小明",
  "gender": "男"
}
```

### 欄位說明
| 欄位 | 類型 | 必填 | 說明 |
|-----|------|------|------|
| `name` | string | ✅ | 姓名 |
| `gender` | string | ✅ | 性別（男/女）|

---

## 成功回應

**HTTP 狀態碼：** 200

```json
{
  "success": true,
  "data": {
    "message": "個人資料更新成功"
  }
}
```

---

## 錯誤回應

### 姓名為空
**HTTP 狀態碼：** 422
```json
{
  "success": false,
  "error": {
    "code": "INVALID_NAME",
    "message": "姓名不可為空"
  }
}
```

### 性別格式錯誤
**HTTP 狀態碼：** 422
```json
{
  "success": false,
  "error": {
    "code": "INVALID_GENDER",
    "message": "性別格式錯誤"
  }
}
```
