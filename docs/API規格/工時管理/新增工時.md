# 新增工時記錄

**端點：** `POST /api/v1/timesheets`  
**權限：** 所有員工

---

## 請求

### Body
```json
{
  "work_date": "2025-11-01",
  "client_id": "12345678",
  "service_id": 3,
  "work_type_id": 1,
  "hours": 8.0,
  "notes": "記帳作業"
}
```

---

## 成功回應

**HTTP 狀態碼：** 201

```json
{
  "success": true,
  "data": {
    "time_log_id": 15,
    "weighted_hours": 8.0,
    "message": "工時記錄新增成功"
  }
}
```
