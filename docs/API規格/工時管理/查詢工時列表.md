# 查詢工時列表

**端點：** `GET /api/v1/timesheets`  
**權限：** 所有員工

---

## 請求

### 查詢參數
| 參數 | 類型 | 必填 | 說明 |
|-----|------|------|------|
| `month` | string | ❌ | 月份（YYYY-MM，預設當月）|
| `user_id` | integer | ❌ | 員工ID（管理員可查詢所有人）|

---
