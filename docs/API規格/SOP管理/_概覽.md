# SOP管理 API 概覽

**API 前綴：** `/api/v1`  
**最後更新：** 2025年10月27日

---

## 端點總覽

| 端點 | 方法 | 權限 | 說明 |
|------|------|------|------|
| `/knowledge/sop` | GET | 所有員工 | 查詢 SOP 列表 |
| `/knowledge/sop/:id` | GET | 所有員工 | 查詢 SOP 詳情 |
| `/knowledge/sop` | POST | 管理員 | 新增 SOP |
| `/knowledge/sop/:id` | PUT | 管理員 | 更新 SOP |
| `/knowledge/sop/:id` | DELETE | 管理員 | 刪除 SOP |
