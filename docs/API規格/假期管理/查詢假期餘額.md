# 查詢假期餘額

**端點：** `GET /api/v1/leaves/balances`  
**權限：** 所有員工

---

## 請求

### 查詢參數
| 參數 | 類型 | 必填 | 說明 |
|-----|------|------|------|
| `year` | integer | ❌ | 年度（預設當年）|

---
