# 新增假期申請

**端點：** `POST /api/v1/leaves/applications`  
**權限：** 所有員工

---

## 請求

### Body
```json
{
  "leave_type_id": 1,
  "start_date": "2025-11-10",
  "end_date": "2025-11-12",
  "total_hours": 24,
  "reason": "家庭事務"
}
```

---

## 成功回應

**HTTP 狀態碼：** 201

```json
{
  "success": true,
  "data": {
    "leave_application_id": 15,
    "status": "pending",
    "message": "假期申請已提交，等待審核"
  }
}
```

---

## 錯誤回應

### 餘額不足
**HTTP 狀態碼：** 422
```json
{
  "success": false,
  "error": {
    "code": "INSUFFICIENT_BALANCE",
    "message": "假期餘額不足"
  }
}
```
