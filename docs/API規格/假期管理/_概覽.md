# 假期管理 API 概覽

**API 前綴：** `/api/v1`  
**最後更新：** 2025年10月27日

---

## 端點總覽

| 端點 | 方法 | 權限 | 說明 |
|------|------|------|------|
| `/leaves/applications` | GET | 所有員工 | 查詢假期申請列表 |
| `/leaves/applications` | POST | 所有員工 | 新增假期申請 |
| `/leaves/applications/:id/approve` | POST | 管理員 | 審核假期 |
| `/leaves/balances` | GET | 所有員工 | 查詢假期餘額 |
