schedule 工時系統
如何記錄客戶工時？
expand_more
1. 進入工時表頁面
2. 選擇員工和月份
3. 點擊「增加工時記錄」
4. 填寫客戶名稱、業務類型和工時類型
5. 在對應日期填入工時數
6. 記得儲存變更
如何申請請假？
expand_more
1. 進入工時表頁面
2. 點擊「增加請假記錄」
3. 選擇假別類型
4. 在對應日期填入請假時數
5. 儲存後通知主管審核
task_alt 任務管理
週期任務如何設定？
expand_more
1. 進入任務頁面，選擇「週期任務」標籤
2. 點擊「新增服務配置」
3. 選擇客戶、服務類型和類別
4. 設定月度執行計畫
5. 系統會自動在每月生成對應任務
工商登記流程怎麼追蹤？
expand_more
1. 進入任務頁面，選擇「工商」標籤
2. 創建新的工商任務
3. 系統會自動建立多階段流程
4. 依序完成各階段並更新狀態
5. 可在時間軸上查看整體進度
assessment 報表分析
如何查看員工工時統計？
expand_more
1. 進入報表頁面
2. 選擇「工時分析」標籤
3. 設定查詢期間和員工
4. 點擊「生成報表」
5. 可匯出Excel進行進一步分析
settings 系統設定
如何新增客戶資料？
expand_more
1. 進入設定頁面
2. 選擇「客戶管理」標籤
3. 點擊「新增客戶」
4. 填寫客戶基本資料
5. 儲存後可在工時表中使用
如何修改密碼？
expand_more
1. 進入設定頁面
2. 選擇「帳號設定」標籤
3. 點擊「修改密碼」
4. 輸入舊密碼和新密碼
5. 確認後立即生效
